Front End Bash Scripting
========================

## Install React on MacOS
```bash
if ! command -v nodejs &>/dev/null
then
    brew install nodejs
else
    echo "nodejs installed already"
fi
```
## Install React on Linux
```bash
if ! command -v nodejs &>/dev/null
then
    apt install nodejs
else
    echo "nodejs installed already"
fi
```
## Install React on Window
```bash

if ! command -v nodejs &>/dev/null
then
    winget install nodejs
else
    echo "nodejs installed already"
fi
```
